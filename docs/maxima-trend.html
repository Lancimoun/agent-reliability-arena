Daily Reliability Trend

The trend dashboard is ready, but no live Maxima imports have been published yet.

Configure the GitHub secret MAXIMA_SYNC_SECRET, then run the Maxima Reliability Trend workflow manually once. After that, it will refresh daily at 09:00 IST.

The workflow stores compact public-safe rows only: score, verdict, pass/warn/fail counts, timestamp, and token-stripped source URL.
